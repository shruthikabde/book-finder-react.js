# 📚 Book Finder React App

A simple React web app to search for books by title using the **Open Library API**.

---

## 🚀 Features
- 🔍 Search books by title
- 🖼️ Displays cover image, author, and publication year
- 📘 Modal popup for more book details (subjects, publisher)
- 🧹 Clear search button
- ⏳ Loading and ❌ error messages
- 💻 Responsive, modern UI

---

## 🧠 Tech Stack
- **Frontend:** React (Create React App)
- **Styling:** Custom CSS
- **API:** Open Library API (no API key required)
